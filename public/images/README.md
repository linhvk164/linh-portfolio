# Image assets

Drop files here; paths are referenced from `data/projects.ts` and components.

## Project folders

| Folder | Use |
|--------|-----|
| `un-habitat/` | UN-Habitat project — `cover.jpg`, section images |
| `folio/` | Folio project |
| `songwriting-app/` | Chordio project (folder name unchanged) |
| `lofu/` | Lofu project |

Suggested naming: `cover.jpg` for grid covers, then descriptive names for case-study images (e.g. `process-01.jpg`).

## General

| Folder | Use |
|--------|-----|
| `general/` | Homepage & site-wide assets (articles, recommendations, profile, etc.) |

Paths in code use `/images/...` (maps to this `public/images/` directory).
